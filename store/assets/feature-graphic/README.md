# Feature graphic

## Wymagania

- format: `JPEG` albo `PNG 24-bit bez alpha`
- rozmiar: `1024 x 500`

## Rekomendacja dla Tyflocentrum

Feature graphic powinien eksponować:

- markę `Tyflocentrum`,
- skojarzenie z podcastami, artykułami i radiem,
- prosty, czytelny motyw bez nadmiaru tekstu.

## Stan projektu

W katalogu jest już robocza propozycja:

- `tyflocentrum-feature-graphic-1024x500.png`

To jest sensowna baza do pierwszego draftu w Play Console, ale przed publikacją warto jeszcze ręcznie ocenić:

- czy ilość tekstu nie jest zbyt duża,
- czy branding jest wystarczająco spójny z docelową ikoną aplikacji,
- czy finalna wersja nie powinna być uproszczona pod kątem sklepowej ekspozycji.

## Czego unikać

- ramek urządzeń,
- zbyt długich haseł marketingowych,
- informacji o rankingu, promocji albo cenie,
- grafiki bardzo podobnej do samej ikony aplikacji.
